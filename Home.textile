h1. Contributing to this wiki

h2. online

If you have a Github account, you can login and edit online. Login is required
for online edit.

h2. with Git

1. First, clone this wiki with the link the the right menu or fetch the lastest
updates from the official wiki (git pull).
2. "Fork the mirror":http://github.com/OfflineIMAP/offlineimap-wiki.
3. Add a reference to your fork: git remote add myfork git@github.com:<username>/offlineimap-wiki.git
4. Edit, edit, edit
5. Commit your changes.
6. Submit a pull requet.
